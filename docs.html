The beginning

How it all started

Movie Buddy started as a personal frustration — too many streaming platforms, too many movies saved in random notes apps, and no way to remember what a friend had recommended weeks ago.

The idea was simple: build one place to track what you've watched, save what you want to watch, and connect with a few trusted friends who share your taste. No ads, no algorithmic noise — just a clean, fast tool for people who actually care about movies.

What began as a wishlist and a "watched" checkbox quickly grew. Friends wanted to send each other recommendations. The AI picks feature arrived after noticing that "liking" certain films consistently predicted what else someone would enjoy. The mood discovery tab came from the simple desire to find the right film for a rainy Sunday afternoon.

Every feature in Movie Buddy exists because it solved a real problem someone had. Nothing was added for the sake of it.

Personal project No ads. Ever. Friends-first design Privacy by default
"The best films don't need a trailer — they need the right friend to say 'you have to watch this.'" — The idea behind Movie Buddy

Built from scratch

No framework. No boilerplate. Every line — from the Node.js server to the service worker cache — was written intentionally. The result is an app that loads fast, works offline, and costs almost nothing to run.

How it evolved

The evolution

From a simple tracker to a full social cinema companion — each phase built on the last.

🎬

Phase 1 — Foundation

Core Movie Tracking

The basics: sign up, search any movie or series from TMDB's catalog, add to wishlist, mark as liked, track what you've watched. Clean auth system with email OTP and password. The first version proved the concept worked.

Search & Discovery Wishlist Watched History Email Auth
👥

Phase 2 — Social Layer

Friends & Recommendations

Added the ability to find friends by handle, send friend requests, and recommend specific movies with a personal note. An activity feed let you see what friends were watching in real time. Push notifications kept everything timely.

Friend Requests Movie Recommendations Activity Feed Push Notifications
🤖

Phase 3 — Intelligence

AI Picks & Mood Discovery

AI-powered recommendations trained on your liked movies — not generic "popular" content but genuinely personalised picks. The Mood tab let users browse 10 emotional categories (Action & Thrills, Romance, Mind Benders, etc.) with language filters for Indian, Korean, Japanese and more.

AI Recommendations 10 Mood Categories Language Filters Surprise Me
📋

Phase 4 — Organisation

Lists, Ratings & Providers

Custom movie lists with public/private toggle. Star ratings (1–5) with personal reviews. Streaming provider info so you always know where to watch. A powerful library editor to reorganise your shelves. Release date reminders with browser push.

Custom Lists Star Ratings Streaming Providers Release Reminders
🔐

Phase 5 — Polish & Privacy

Privacy Controls & Profiles

Gated friend profiles — before a request is accepted, you see a lock. After acceptance, friends control exactly what's visible: wishlist, liked movies, watch history. Profile insights with genre breakdowns, watch stats, monthly charts. The accept/decline UX was redesigned to be visually clear and satisfying.

Friend Privacy Toggles Profile Insights Genre Stats Monthly Charts
🖥️

Phase 6 — Security & Control

Device Management & Session Security

Multi-device support with a 3-device limit per account. Each login session tracks your device (browser + OS) so you always know where you're logged in. The Profile tab now shows an Active Devices panel — see every live session, spot the one you're on ("This device"), and sign out individual devices or all others at once. When you log in on a 4th device, the oldest session is silently evicted so you're never locked out.

How it's built

System architecture

Three layers. Zero unnecessary dependencies. Everything from auth to push notifications runs on a single Node.js process.

Client Layer Browser · PWA · Service Worker
index.html + app.js Vanilla JS SPA — no framework, no build step
Service Worker (sw.js) Offline cache · Push notifications · Background sync
Google Fonts + CSS Tokens Cinema pro dark theme · Responsive layout
Canvas API Shareable movie cards generated client-side
HTTP / HTTPS — REST API calls + static file serving
Server Layer Node.js 24 · Built-in http module · No Express
Auth RoutesEmail OTP · Phone OTP · Google OAuth · Session cookies
Profile RoutesSettings · Avatar · Privacy · Stats · Sync
Friends RoutesRequests · Overlap · Profiles · Recommendations
Catalog RoutesTMDB proxy · Search · Trending · Upcoming · Providers
AI RoutesPersonalised picks based on liked movies
Activity RoutesFeed · Reactions · Comments · Notifications
Push RoutesVAPID keys · Subscriptions · Release reminders scheduler
Lists & AdminCustom lists · Stats dashboard · DB migrations
LibSQL protocol (HTTPS) — stateless per-request connections
Database Layer LibSQL / Turso — SQLite-compatible
usersid · handle · name · email · phone · password_hash (never plaintext)
profilesJSON blob — wishlist · liked · watched · friends · privacy · inbox
sessionstoken · user_id · expires_at · device_info — HttpOnly cookies, max 3 per user
activity_feed+ reactions + comments — social layer tables
user_lists+ list_items — custom curated collections
contact_messagesBug reports · Feature requests from docs page

Getting started

How to use Movie Buddy

From first login to your first friend recommendation — here's how everything connects.

01
🔑

Create your account

Sign up with email (OTP verification), phone number, or your Google account. Set a unique @handle — friends will use this to find you.

02
🎬

Build your library

Search any title. Add to Wishlist (want to watch), Like (already love it), or Now Watching (currently in progress). The more you like, the smarter the AI gets.

03
🤖

Unlock AI picks

Like 5+ movies, save your starter shelf. Your AI Picks tab activates with personalised recommendations — not crowd favourites, but picks matched to your exact taste.

04
🎭

Explore by mood

Not sure what to watch? Hit the Mood tab. Pick a vibe — Thrills, Romance, Mind Bender, Laugh Out Loud — and filter by language. Hit Surprise Me for a random pick.

05
👥

Add friends

Search by @handle in the Friends tab or the topbar. Send a request — once accepted, you'll see each other's activity, profiles, and recommendations.

06
📨

Send recommendations

On any movie card, hit "Send to friend", pick who to send it to, and add a personal note. They'll see it in their recommendations inbox and get a push notification.

07
⭐

Rate what you watch

After marking something watched, rate it 1–5 stars. 4★ and above boosts that film's signal in your AI recommendations. Your average rating and stats update in real time.

08
🔔

Set release reminders

Add upcoming films to your Reminders shelf. Enable browser push, set your timezone and lead time — Movie Buddy will ping you when the release is near.

09
🖥️

Manage your devices

Open the Profile tab and scroll to Active Devices. You'll see every device currently signed in (browser + OS), which one is the current device, and when each session was created. Sign out any individual device or hit "Sign out other devices" to clear all others at once. You can be logged in on up to 3 devices simultaneously.
